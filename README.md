# Automação para envio de mensagens pelo WhatsApp

## Entendendo a Automação 
A automação precisa necessáriamente de uma lista, independente do formato, ela irá ler todos os dados, até finalizar. 

Sendo assim, o ciclo dela é Abre navegado -> Acessa o Web WhatsApp -> Envia a mensagem -> Fecha o navegador, até acabar a lista. 

## Bibliotecas
Necessário executar o requirements.txt ou baixar diretamente por *pip install openpyxl, pyautogui, pillow*, demais bibliotecas são nativas.
